# Contribuyendo a Dami Fast Food

¡Gracias por tu interés en colaborar! Para mantener el proyecto organizado, sigue estas pautas:

## ¿Cómo colaborar?
1. **Reportar errores:** Si encuentras un fallo en el menú digital o en el código, abre un "Issue".
2. **Sugerir mejoras:** Si tienes ideas para nuevos platos o funciones, también puedes usar los "Issues".
3. **Enviar cambios:** Por ahora, consulta conmigo antes de enviar un Pull Request para asegurar que el diseño se mantenga uniforme.

¡Tu ayuda es bienvenida!
